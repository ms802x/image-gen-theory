# Notes

Use this folder to track reading notes, notebook source surveys, and paper-to-code mappings.

Suggested files:

- `readings.md`: short notes from papers, tutorials, and demos.
- `notebook_sources.md`: public notebooks/repos inspected before writing each homework notebook.
- `qwen_mapping.md`: map the final tiny model to Qwen-Image components and gaps.

For each source, record:

- link
- topic
- what it teaches
- what is confusing or missing
- whether it was run, skimmed, or deeply studied
- whether code was copied, adapted, or only used conceptually
